..  py:data:: hexagrams(dict)

    A dict of the brief information on the 64 hexagrams in the form ::

        {number: (hexagram_symbol, chinese_character, english_translation)}

    For example, ::

        hexagrams = {
            1:  ('䷀', '乾',   'Force'),
            2:  ('䷁', '坤',   'Field'),
            3:  ('䷂', '屯',   'Sprouting'),
            4:  ('䷃', '蒙',   'Enveloping'),
            .
            .
            .
            64: ('䷿', '未濟',  'Not Yet Fording')
        }
